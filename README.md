# Full Stack AI Learning Program

This repository tracks my 20-week full stack development with AI learning journey.

## Goal

My goal is to become capable of building, explaining, and deploying practical full stack AI applications that solve real business workflow problems.

I am focusing on:

* Developer setup, terminal, Git, and GitHub
* HTML, CSS, responsive design, and Tailwind CSS
* JavaScript fundamentals
* React and TypeScript
* Node.js and Express
* PostgreSQL and Prisma
* Authentication and basic security
* Next.js, Supabase, and Vercel
* OpenAI API integration
* Embeddings, vector search, and RAG
* Testing, CI, and deployment
* Portfolio-ready documentation

## Data Safety Rule

This course uses fictional and dummy data only.

I will not upload or commit:

* Real employer documents
* Client data
* Credentials
* API keys
* Security reports
* HR records
* Financial records
* Internal company workflows
* Real contracts
* Real audit findings
