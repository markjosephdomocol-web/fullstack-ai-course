# Command Practice

## Folder navigation

```bash
pwd
ls
cd ..
cd folder-name
```

## File and folder creation

```bash
mkdir example-folder
New-Item example-file.md
```

## Git basics

```bash
git status
git add .
git commit -m "docs: add week 1 notes"
git log --oneline
```
